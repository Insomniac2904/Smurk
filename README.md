## Installation

To install, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/Insomniac2904/Smurk.git
    ```
2. Navigate to the project directory:
    ```bash
    cd Smurk
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```
4. To Build :
   ```
   npm run build:(your os)
   ```
